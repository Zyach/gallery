## Phase 3: HTTP API Hardening & Testing (Days 3–4)

**Goal:** Ensure HTTP API still works; fix any inference callback breakage.

### 3.1 Validate HTTP Service Compatibility

**Test Matrix:**
| Scenario | Test | Expected Outcome |
|----------|------|------------------|
| Service startup | Start via SettingsDialog | HTTP server listens on 127.0.0.1:9006 |

| Tool calls | POST with tools array | Synthesized tool call in response |
| Model selection | POST with model param | Correct model name in response |

| Payload limits | POST >512KB | Returns 413 Payload Too Large |

**Code to verify:**
